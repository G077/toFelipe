 Esse trabalho lê textos com characters da tabela ascii pois a funcao open() nẫopermite outras encodações,então textos em ingles sem acentos portugueses são melhores para esse projeto.


Para compilar o projeto execute o arquivo make.sh;

 Pesquisa sequencial == pseq
 Pesquisa Binaria == pbinaria 
 Arvore == arvore
 Hash aberto == haberto
 Hash fechado == hlista

se nao achar nao mostrara no output a quantidade vezes que repitou e as interaçoes somente o tempo de execuçao;
